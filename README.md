# QueBox

## What it is?

QueBox is a web application, the purpose is to make 3d boxes that can have different dimensions and images in any of his horizontal layers.

The user can create an account and store designs of boxes in his account.

The box can have:
  - large
  - width
  - height
  - 4 layers to add n cuantity of images

The plot can be interactive and downloable.
